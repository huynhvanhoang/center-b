# Set Scale

대상의 Scale 값을 설정합니다.
## Parameter

| **이름**      | **설명**                                 |
|-------------|----------------------------------------|
| Game Object | Scale을 변경할 Game Object를 설정하세요.         |
| scale       | 설정하는 Game Object의 값으로 Rotation을 변경합니다. |


## Tip

1. Game Object 구조에 따라 World 와 Local 이 상이한 결과를 보일 수 있습니다.
