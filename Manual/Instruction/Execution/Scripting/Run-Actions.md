# Run Actions

Performer를 실행하면 연결된 Performer를 실행합니다.


## Parameter

| **이름**    | **설명**                    |
|-----------|---------------------------|
| Performer | 실행할 Performer 오브젝트를 연결합니다 |


## Tip

1. Trigger 하위에 직접 연결된 Performer는 실행할 수 없습니다.
2. Performer Component으로 구성된 객체만 Run Performer의 영향을 받습니다.
