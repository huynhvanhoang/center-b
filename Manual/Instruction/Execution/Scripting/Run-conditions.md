# Run conditions
대상 오브젝트 내의 Condition 컴퍼넌트를 실행하도록 합니다. 


## Parameter

| **이름**    | **설명**                                    |
|-----------|-------------------------------------------|
| Condition | 대상 Condition 컴포넌트를 가지고 있는 게임 오브젝트를 설정합니다. |



## Tip

1. 스크립트의 구조를 모듈화 하여 사용하고자 하는 경우 Trigger 컴포넌트에 직접 스크립트를 구축하는것 보다 효과적일 수 있습니다.
