# Toggle Active

Performer를 실행하면 Active의 활성 상태를 전환합니다.  
Active가 활성(True) 상태라면 비활성(False) 상태로 전환하고, 비활성(False) 상태라면 활성(True) 상태로 전환합니다.


## 파라미터

| **이름** | **설명**                   |
|--------|--------------------------|
| Target | Active 상태를 전환할 대상을 설정합니다 |
