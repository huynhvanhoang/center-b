# Item Destroy

Performer를 실행하면 아이템을 소지한 캐릭터(플레이어)에게서 아이템을 삭제합니다  
드롭이 실행될 때 HUD UI Button에서 아이템 아이콘도 같이 삭제됩니다.


## Parameter

| **이름** | **설명**                           |
|--------|----------------------------------|
| Target | 아이템을 삭제 할 대상을 설정합니다.             |
| Slot   | 삭제할 슬롯 번호(HUD UI Button)를 입력합니다. |


## Tip

1. Item Destory은 설정한 아이템을 제거하는 기능입니다.
2. 드롭 기능을 제공하지 않습니다.
