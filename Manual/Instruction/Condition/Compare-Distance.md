# Compare Distance

두 오브젝트의 거리를 비교하여 일치할 때 true 를 반환합니다

## Parameter

| **이름**         | **설명**                            |
|----------------|-----------------------------------|
| First Vactor3  | 거리를 구할 첫번째 오브젝트의 Vactor3 값을 설정합니다 |
| Second Vactor3 | 거리를 구할 두번째 오브젝트의 Vactor3 값을 설정합니다 |
| Compare Float  | 비교할 Float 값을 설정합니다                |
