# String

Variable에 String(문자) 값을 저장합니다

## Parameter

| **이름**               | **설명**               |
|----------------------|----------------------|
| Value                | String(문자) 값을 설정합니다. |


## Tip
1. 스트링을 이용한 값을 설정하거나, 주고 받을 경우에 사용하는 값입니다.
2. variable을 Instruction에서 참조할 경우 Key값을 동일하게 맞춰야 합니다.
3. key 값이 맞지 않는 경우 스크립트는 무시됩니다.
