# Game Object

Variable에 Game Object 값을 저장합니다

## Parameter

| **이름**               | **설명**               |
|----------------------|----------------------|
| Value                | Game Object값을 설정합니다. |


## Tip
1. 게임 오브젝트를 설정하고, 값을 변경하거나 참조할 수 있습니다.
2. 게임 오브젝트에는 "Effect"도 포함됩니다.
3. variable을 Instruction에서 참조할 경우 Key값을 동일하게 맞춰야 합니다.
4. key 값이 맞지 않는 경우 스크립트는 무시됩니다.
